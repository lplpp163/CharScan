# CarbonLens（前身 CharScan）

> 拍一張照片，便宜又非破壞地量出植物固了多少碳。
> Snap a photo, non-destructively measure how much carbon a plant holds.

**2026 Net Zero Tech 淨零排放科技國際競賽**提案方向總覽 — Sustainability 永續組（碳移除 / MRV）。

便宜影像＋環境時序（溫度/光照） → **AI 估生物量 → 每株總碳量** → 少量 CHN＋乾重校正 → **低成本碳量測 (MRV)**。

## 核心定位（純萵苣量碳尺版 · 無生物炭 / 無燃燒）

- **萵苣＝模式生物（試驗載體）**，不是碳匯標的。像生物學用大腸桿菌/阿拉伯芥——用快速便宜可重現的作物把「量碳技術」調準。**從不主張萵苣固碳。**
- **我們賣「量碳的尺(MRV)」，不種碳、不儲碳、不賣碳權、不燃燒。** 講固碳效益/量固碳量＝技術（工研院、台泥都這樣講）；宣稱拿可交易碳權＝金融商品，要方法學認證——我們不跨那條線。儲存碳的事交給大自然（森林、土壤）與專案開發商。
- **可遷移到真實碳庫**：森林立木生物量、土壤碳活動數據、生態復育——這些需要便宜的非破壞 MRV，我們只負責「量」。

## 量測目標：每株總碳量（不是碳濃度）

- 植物乾物質的**碳分率(%C)幾乎是常數（約 45%）**，跨環境變動極小——預測它沒有意義。
- 真正會變、也真正值得量的是 **每株總碳量 ＝ 乾生物量 × 碳分率**，變動幾乎全來自**生物量與乾物率（含水量）**。
- 因此模型用 **影像（冠層面積/綠度）＋環境時序（累積光照 DLI／積溫）** 估生物量，再乘碳分率得總碳。
- 真值＝ **CHN 測碳分率(%C) × 乾重**。⚠️ 採收時必須**同時量鮮重與乾重**（只有 %C 會拿到一個無用的常數）。

## 硬數字怎麼來（不靠生物炭）

- **報名前（零硬體）**：用公開「作物光譜↔生物量/碳」資料集跑概念驗證 R²。
- **進行中**：已送萵苣樣品做 CHN 元素分析（等結果）；自種分批採收 → 控光拍照＋環境時序 → 同時量鮮重/乾重 → 乾重×CHN(%C)＝每株總碳量真值 → 報自有 R²/RMSE。
- 唯一硬依賴＝CHN（少量送驗校正）；**完全不需要爐子/熱解/燃燒**。手持 NIR 為選配加分，主打影像＋環境時序。

## 這個 repo 是什麼

`index.html` 是給組員的**單頁方向總覽**，整合：核心轉念（萵苣＝模式生物、AI＝量碳尺、量測 ≠ 碳權）、量測目標（每株總碳）、評審 5 點、TWN15 通俗解說與借鏡手法、進度誠實盤點、定位邏輯、新穎性誠實定位、材料 × 資工分工、設備清單與本週確認清單、官方摘要表六欄企劃骨架。直接用瀏覽器開啟即可。

## 紅線（務必守住）

草本本體吸的碳屬 biogenic 短循環，採收/被吃後幾週回大氣，淨庫存變化≈0 → **不可主張萵苣固碳/碳權**。CarbonLens 不去當碳庫，而是去當「量任何碳庫的尺」。
