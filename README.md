## 核心定位

- **萵苣＝快速驗證作物**，不是碳匯標的。用生長週期短、便宜、可重現的作物把量測方法調準。不主張萵苣固碳。
- **做的是量測 (MRV)，不是碳權。** 量固碳量是技術；可交易碳權是金融商品，需方法學認證，由下游碳庫與專案開發商處理。
- **可遷移到真實碳庫**：森林立木生物量、土壤碳、生態復育——這些需要便宜的非破壞量測。

## 量測目標：每株碳量（不是碳濃度）

- 植物乾物質的碳分率（%C）接近常數（約 47%；IPCC 2006 預設 0.47 tC/t；Ma et al. 2018 在 4,318 物種落在 45.0–47.9%），預測它沒有意義。
- 真正會變、值得量的是 **每株碳量 = 乾生物量 × 0.47**，變動幾乎全來自生物量。
- 模型用影像（冠層面積／株高／綠度）＋環境時序估生物量，少量烘乾乾重校正「影像→生物量」這條關係，碳量再由 0.47 換算，不需對每株做元素分析。

## 目前成績

自有萵苣資料集（569 張、第 7–35 天），每株乾重模型用其中有單顆乾重的 69 株訓練與驗證。側拍結構特徵＋累積環境，留一日交叉驗證 **R²≈0.61、RMSE≈0.44 g**。實測結果，非模擬。

## 紅線（務必守住）

草本本體吸的碳屬短循環，採收／被吃後幾週回大氣，淨庫存變化≈0，不可主張萵苣固碳或碳權。LeafCarbon 不當碳庫，而是量任何碳庫的尺。
