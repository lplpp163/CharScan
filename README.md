# CarbonLens（前身 CharScan）

> 拍一張照片，便宜又非破壞地量出植物固了多少碳。
> Snap a photo, non-destructively measure how much carbon a plant holds.

**2026 Net Zero Tech 淨零排放科技國際競賽**提案方向總覽 — Sustainability 永續組（碳移除 / MRV）。

便宜影像 / 手持 NIR → **AI 非破壞預測碳量** → 少量元素分析(CHN)校正 → **低成本碳量測 (MRV)**。

## 核心定位（純萵苣量碳尺版 · 無生物炭 / 無燃燒）

- **萵苣＝模式生物（試驗載體）**，不是碳匯標的。像生物學用大腸桿菌/阿拉伯芥——用快速便宜可重現的作物把「量碳技術」調準。**從不主張萵苣固碳。**
- **我們賣「量碳的尺(MRV)」，不種碳、不儲碳、不賣碳權、不燃燒。** 講固碳效益/量固碳量＝技術（工研院、台泥都這樣講）；宣稱拿可交易碳權＝金融商品，要方法學認證——我們不跨那條線。儲存碳的事交給大自然（森林、土壤）與專案開發商。
- **可遷移到真實碳庫**：森林立木生物量、土壤碳活動數據、生態復育——這些需要便宜的非破壞 MRV，我們只負責「量」。

## 硬數字怎麼來（不靠生物炭）

- **報名前（零硬體）**：用公開「作物光譜↔生物量/碳」資料集跑概念驗證 R²。
- **入圍後**：自種萵苣分批採收 → 控光拍照＋手持 NIR → 少量 CHN 測碳含量真值 → 報自有 R²/RMSE。
- 唯一硬依賴＝CHN（少量送驗校正）；**完全不需要爐子/熱解/燃燒**。

`index.html` 是給組員的**單頁方向總覽**，整合：核心轉念（萵苣＝模式生物、AI＝量碳尺、量測 ≠ 碳權）、評審 5 點、TWN15 通俗解說與借鏡手法、進度誠實盤點、定位邏輯、新穎性誠實定位、材料 × 資工分工、設備清單與本週確認清單、官方摘要表六欄企劃骨架。直接用瀏覽器開啟即可。

## 紅線（務必守住）

草本本體吸的碳屬 biogenic 短循環，採收/被吃後幾週回大氣，淨庫存變化≈0 → **不可主張萵苣固碳/碳權**。CarbonLens 不去當碳庫，而是去當「量任何碳庫的尺」。
