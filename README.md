# CharScan

> 拍一張照片，驗證一塊炭鎖住了多少碳。
> Snap a photo, verify how much carbon a piece of char locks away.

**2026 Net Zero Tech 淨零排放科技國際競賽**提案方向總覽 — Sustainability 永續組（碳移除 / MRV）。

蔬果廢料 → 熱解製生物炭 → **AI 低成本驗證 (MRV)** → 可交易碳權。

## 這個 repo 是什麼

`index.html` 是給組員的**單頁方向總覽**，整合：

- 背景：什麼是固碳、問題在哪（30 秒看懂）
- 選題邏輯：為什麼選生物炭＋AI（四個限制條件對比）
- 新穎性：參考 20 張歷屆得獎海報歸納的「得獎公式」
- 高勝率訂題法
- 技術方案與材料 × 資工分工
- 設備清單與材料組員首要確認清單
- 沒設備也能用公開資料集做的退路
- 對應官方摘要表六欄的完整中文企劃骨架

直接用瀏覽器開啟 `index.html` 即可閱讀。
